# NewsApp + React + Material UI

In this app articles are fetched from News API. There are several functionalities such as
- searching,
- infinite scrolling 
- dividing articles into categories.
> General, latest news and search bar are displayed by default.<br>
General news can be replaced with news from some other category with click on a button while latest news are organized
by the time they got published + they are infinite scrollable. <br>
News are searchable - user can type word (or couple of words) and app will provide articles with those words in them. <br>
The app is also built to support smaller screen devices - mobile phones. The layout differs depending on whether the user is on desktop or mobile
